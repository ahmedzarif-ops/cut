# Paid Media Engine — Learnings

The running registry of paid-media runs. Append one entry per run so the next run starts
from what you already learned.

## How to use this file

After each run, append an entry under Runs with: the date, the brief (offer, segment,
channels), what the strategy bet on, what copy shipped, the compliance result, what
worked or surprised, and what to change next time. Keep entries short and skimmable.
Cross-reference the `market-radar` output for demand context.

Entry shape:

```
### YYYY-MM-DD — <short title>
- Brief: <offer / segment / channels>
- Strategy bet: <the single biggest bet>
- Shipped: <what copy / how many campaigns / which platforms>
- Compliance: <PASS / flags fixed>
- Worked / surprised: <notes>
- Change next time: <notes>
```
