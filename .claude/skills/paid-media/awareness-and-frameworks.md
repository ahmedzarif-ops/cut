# Awareness Stages and Copywriting Frameworks

Two tools: the awareness matrix tells the `growth-strategist` what message a buyer is
ready for; the framework selector tells the `paid-media-copywriter` how to structure
the copy. Match the message to the stage, then pick the framework for the stage,
platform, and format. See `creative-angles.md` for the angle library.

## Awareness-stage messaging matrix

Eugene Schwartz's five stages. The belief is the buyer's starting thought; the message
job is what the copy must do; then the best CTA/offer. Tailor the belief wording to
your ICP (`company.yml -> icp`) — the phrasings below are generic scaffolding.

### Unaware
- Belief: "Business is just uneven sometimes. That is how it goes."
- Message job: show that the missed outcome is usually not a quality problem — it is a
  visibility, proof, clarity, response, or follow-up problem.
- Best CTA / offer: the free diagnostic.

### Problem-Aware
- Belief: "I know I need more leads / visibility / a better system."
- Message job: name the specific leak (invisible in search, weak proof/reviews, poor
  response, follow-up gap, confusing site, reliance on a rented/expensive channel).
- Best CTA / offer: the free diagnostic or the relevant guide.

### Solution-Aware
- Belief: "I need [a category of solution] or outside help."
- Message job: contrast a scattered single tactic with a complete owned system.
- Best CTA / offer: the diagnostic or the entry paid offer.

### Product-Aware
- Belief: "I am comparing tools, agencies, and systems."
- Message job: explain the exact workflow, what is included, what happens over time,
  how approval works, how it connects to what they already use, and why it is different
  from renting an agency or a lead source.
- Best CTA / offer: the entry offer, a live sample, a demo, or a consultation.

### Most-Aware
- Belief: "I know this company and am considering buying."
- Message job: resolve price, risk, technical fear, setup time, quality, control,
  cancellation, refund, and whether it applies to their situation.
- Best CTA / offer: buy now, subscribe, book a call, or an authentic launch offer.

## Copywriting frameworks

Pick by audience state, format, platform, and offer. One-line definitions:

- PAS: Problem -> Agitate -> Solution.
- PASO: Problem -> Agitate -> Solution -> Outcome.
- AIDA: Attention -> Interest -> Desire -> Action.
- BAB: Before -> After -> Bridge.
- FAB: Features -> Advantages -> Benefits.
- 4Ps: Promise -> Picture -> Proof -> Push.
- QUEST: Qualify -> Understand -> Educate -> Stimulate -> Transition.
- PASTOR: Problem -> Amplify -> Story/Solution -> Transformation -> Offer -> Response.
- ACCA: Awareness -> Comprehension -> Conviction -> Action.
- 4Us: Useful -> Urgent -> Unique -> Ultra-specific (best for headlines).
- SLAP: Stop -> Look -> Act -> Purchase (best for scroll-stopping short social).
- PPPP: Picture -> Promise -> Prove -> Push.
- Claim -> So What -> Business Consequence.
- Hook -> Insight -> Mechanism -> Proof -> CTA.
- Hook -> Problem -> Demo -> Proof -> Offer.
- Problem -> Cost of Inaction -> New Mechanism -> Proof -> Action.
- Contrast -> Consequence -> System -> CTA.

## Framework selector matrix

| Framework | Best awareness stage | Best platform | Ideal length | Ideal CTA | Proof requirement | Risks to avoid |
|---|---|---|---|---|---|---|
| PAS / PASO | Problem-Aware | Paid-social primary text, Reddit | Short to medium | Run the diagnostic | Light (the pain is self-evident) | Over-agitating; sounding shaming |
| BAB | Problem to Solution-Aware | Paid social, email | Medium | See the plan / start free | The "after" must be honest, not guaranteed | Implying guaranteed outcomes |
| 4Ps / PPPP | Solution to Product-Aware | Landing match, email, paid social | Medium | Buy the entry offer / subscribe | Real proof in the Prove step | Weak or fabricated proof |
| AIDA | Problem to Solution-Aware | Paid social, video | Medium | Learn more / start free | Moderate | Generic attention hooks |
| QUEST | Product-Aware | Search landing, long-form | Long | Demo / consult | Strong (qualify then educate) | Burying the qualify step |
| PASTOR | Solution to Most-Aware | Video, webinar, email | Long | Subscribe / book a call | Story + real proof | Story drift; fabricated story |
| 4Us | any | Headlines, RSA headlines | Very short | n/a (headline) | n/a | Vague, non-specific headlines |
| SLAP | Unaware to Problem-Aware | TikTok, Reels, Shorts | Very short | Comment / link in bio | Minimal | Clickbait that misleads |
| Claim -> So What -> Consequence | Solution to Product-Aware | Professional network, Search | Short to medium | Consult / demo | The consequence must be defensible | Unsupported business-impact claims |
| Hook -> Insight -> Mechanism -> Proof -> CTA | Solution to Product-Aware | Video, long paid social | Medium to long | Start free / demo | Real mechanism + proof | Hand-wavy mechanism |
| Hook -> Problem -> Demo -> Proof -> Offer | Product-Aware | Video, screen-rec | Medium | Start free / subscribe | The demo is the proof (label samples) | Faking the demo |
| Problem -> Cost of Inaction -> New Mechanism -> Proof -> Action | Solution to Product-Aware | Search landing, email | Medium | Subscribe / consult | Honest cost framing | Misleading cost stats |
| Contrast -> Consequence -> System -> CTA | Problem to Solution-Aware | Paid social, professional network | Medium | Run the diagnostic | The contrast must be fair | Unfair competitor comparisons |

Default starting picks: paid-social cold = PAS or Contrast -> Consequence -> System ->
CTA to the diagnostic; Search = 4Us headlines + Claim/So-What in descriptions; Video =
Hook -> Problem -> Demo -> Proof -> Offer; professional network = Claim -> So What ->
Consequence for the high-touch offer.
