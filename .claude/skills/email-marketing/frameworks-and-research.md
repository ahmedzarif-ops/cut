# Copywriting Frameworks and Research Methodology

Two tools: the framework selector tells the copywriter how to structure an email; the research
methodology grounds the craft. Pick the framework by awareness stage, lifecycle stage, format,
and offer.

## Framework selector

One-line definitions, then the selector matrix.

- PAS: Problem -> Agitate -> Solution
- PASO: Problem -> Agitate -> Solution -> Outcome
- AIDA: Attention -> Interest -> Desire -> Action
- BAB: Before -> After -> Bridge
- 4Ps: Promise -> Picture -> Proof -> Push
- QUEST: Qualify -> Understand -> Educate -> Stimulate -> Transition
- PASTOR: Problem -> Amplify -> Story/Solution -> Transformation -> Offer -> Response
- ACCA: Awareness -> Comprehension -> Conviction -> Action
- PPPP: Picture -> Promise -> Prove -> Push
- Claim -> So What -> Business Consequence
- Hook -> Insight -> Mechanism -> Proof -> CTA
- Problem -> Cost of Inaction -> New Mechanism -> Proof -> Action
- Story -> Lesson -> Practical Action -> Next Step
- Observation -> Diagnosis -> Fix -> Checklist
- Result -> Reason -> Roadmap -> CTA

### Selector matrix

| Framework | Awareness stage | Lifecycle stage | Ideal length | Best use | Proof requirement | Misuse risk |
|---|---|---|---|---|---|---|
| PAS / PASO | Problem-Aware | nurture, abandonment | short-medium | name a recognizable leak | light (pain is self-evident) | over-agitating, shaming |
| BAB | Problem-to-Solution | welcome, upgrade | medium | the execution-gap reframe | the "after" must be honest | implying guaranteed outcomes |
| 4Ps / PPPP | Solution-to-Product | entry/core-offer interest | medium | make the offer concrete | real proof in the Prove step | weak or fabricated proof |
| QUEST | Product-Aware | core-offer interest, consult | long | qualify then educate | strong | burying the qualify step |
| PASTOR | Solution-to-Most-Aware | upgrade, consult | long | founder-led story + offer | story + real proof | fabricated story |
| Observation -> Diagnosis -> Fix -> Checklist | Problem-Aware | results nurture, welcome | medium | the value-first teaching email | the fix must work | hand-wavy fix |
| Story -> Lesson -> Practical Action -> Next Step | Unaware-to-Problem | newsletter, welcome | medium | build trust + capability | minimal | story drift |
| Result -> Reason -> Roadmap -> CTA | Solution-to-Product | results nurture, onboarding | medium | turn a diagnostic result into a plan | the result must be real (the user's score) | implying a result that did not run |
| Claim -> So What -> Consequence | Solution-to-Product | consult, upgrade | short-medium | the business-impact email | defensible consequence | unsupported impact claims |
| Hook -> Insight -> Mechanism -> Proof -> CTA | Solution-to-Product | core-offer interest | medium-long | explain the system | real mechanism + proof | hand-wavy mechanism |

Default starting picks: welcome = Story->Lesson->Practical Action->Next Step; results nurture =
Observation->Diagnosis->Fix->Checklist or Result->Reason->Roadmap->CTA; entry/core-offer
abandonment = PAS or 4Ps; upgrade = BAB or PASTOR; consult = QUEST or Claim->So What->Consequence.

## Research methodology (labeled, do not over-claim)

Synthesize these into an audience-specific approach. Label each as evidence-based,
practitioner-led, or proprietary; do not repeat simplistic or unverified origin claims.

- Claude Hopkins - measurable direct response, reason-why copy (practitioner-led, historical).
- John Caples - headlines, specificity, testing (practitioner-led).
- Eugene Schwartz - buyer awareness + market sophistication (practitioner-led framework).
- David Ogilvy - research-led messaging and proof (practitioner-led).
- Gary Bencivenga - proof, credibility, specificity (practitioner-led).
- Robert Cialdini - authority, social proof, reciprocity, commitment, genuine scarcity
  (evidence-based, peer-reviewed; use ethically, never manufacture scarcity).
- Joanna Wiebe / Copyhackers - Voice-of-Customer research, message hierarchy, objection mining
  (practitioner-led method).
- Val Geisler - lifecycle, onboarding, retention, customer marketing (practitioner-led).
- Chad S. White - email strategy, customer-centric messaging, measurement (practitioner-led).
- Peep Laja / CXL - hypothesis-led CRO (practitioner-led, experimentation-oriented).
- Ronny Kohavi, Diane Tang, Ya Xu - trustworthy online experimentation (evidence-based).
- Jobs to Be Done / Bob Moesta - trigger, desired progress, anxieties, habits, switching
  (practitioner-led framework).
- April Dunford - positioning vs competitive alternatives (practitioner-led).

VoC discipline: pull the real buyer language and objections from your ICP research
(`company.yml` → `icp.research_doc`, its raw-owner-voice / verbatim section) rather than inventing
phrasing. Honor its reliability flags.
